Automation Insight: Three tasks. Three modules. Three concerns separated cleanly — create the VLAN, configure the ports, assign the VLAN to the ports. The loop: keyword removed duplication across ports. The {{ variable }} removed duplication across switches. This is how automation scales: data changes, logic doesn't.
