The Terraform Workflow
Write
1
Define resources
in .tf config files.
HCL is readable by
humans & machines.
main.tf
variables.tf
outputs.tf
terraform init
2
Download provider
plugins.
Run once per project
or new provider.
$ terraform init
terraform plan
3
Preview every change
before it happens.
Shows +add /~change
/−destroy counts.
$ terraform plan
terraform apply
4
Create or update
real infrastructure.
Confirm with 'yes'
after reviewing plan.
$ terraform apply
terraform destroy
5
Remove everything
Terraform created.
Correct order,
nothing left behind.
$ terraform destroy
The write → plan → apply cycle repeats with every change.
terraform destroy is a separate teardown operation, not a routine next step.
